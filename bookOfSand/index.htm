The book of sand

An alternative way of reading The Book of Sand, a short story by Jorge Luis Borges.

This program relies on a cellular automata where each cell simulates a grain of sand.

instructions

  • • Point and press mouse to drop some sand
  • • Hold 'shift' key while pressing mouse to remove sand

hints

  • • If at some point you can't see text anymore: drop some sand near one of the horizontal sides to create a friendly slope that will bring a new line in
  • • Press 'x' key to wipe-out all the sand

acknowledgments

  • • Thanks to Maximus Clarke, for his hypertext puzzle.
  • • Thanks to Norman Thomas di Giovanni, for his translation to English.

requirements

  • • Decent computer
  • • Modern browser, with limited number of hungry tabs

history

  • • The first iteration of this program was a Java applet created for the Concept.Live/BD4D event #3 that took place in Tel-Aviv in 2004.
  • • Later, it served as the base for developing mobile sand, an innovative game for (pre smartphone) mobiles.
  • • At some point, the Java technology became obsolete in the browser.
  • • Nowadays, the program has been remastered and runs within the browser.
