The war of the words

Two armies clash on the semantic battlefield. Each camp is composed of groups of elevated keywords. Each group is firing at one or more groups from the opposite camp. Each time a keyword is hit, its elevation decrease.

Fire will cease only after one camp has been wiped off the face of the page...

acknowledgments

  • • Based on Peter Beinart's article for the Washington Post.

requirements

  • • Decent computer
  • • Modern browser, with limited number of hungry tabs

history

  • • The first iteration of this program was a Java application running from the browser. As a slashrun, it was possible to edit the content prior to running (choosing your own text and which keywords will be part of the conflict...)
  • • At some point, the Java webstart technology became obsolete.
  • • Nowadays, the program has been remastered and runs within the browser. However, the text and keywords are not editable anymore.
